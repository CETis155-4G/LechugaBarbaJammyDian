Lenguaje HTML



HTML es un programa de tipeado con la estructura de un lenguaje de programacion, pero que en sí, no funciona como uno. Tal como su nombre indica Hypertext Marking Language o en español Lenguaje Marcado de Hipertexto solo es un idioma de marcado. Su versión más actualizada que es HTML5 es una versión estándar del programa, con estandar quiere decir que es sólo el patrón de referencia; sin embargo, no es un programa profesional. Se maneja a base de etiquetas y sirve principalmente para el diseño de paginas web junto con CSS y Javascript.

El significado de sus siglas refiere a que, en primera, HyperText que es un "hipertexto", es basicamente un texto que enlaza a otros contenidos ya sean, sitios web, videos, citas. Markup son las etiquetas con las que hace funcionar el programa, cuyas etiquetas comienzan un parrafo, construyen ese parrafo y se cierra. Y finalmente Language es porque, sencillamente, HTML es un lenguaje.

HTML maneja un "esqueleto" o estructura que se conforman mediante el siguiente orden de etiquetas basico: html, head, body, footer.

Lenguaje CSS

CSS (Cascading Style Sheets) es un lenguaje modelado o un lenguaje de diseño gráfico que en español refiere a "hojas estilo cascada" y se le denomina así porque las caracteristicas se aplican de arriba a abajo. Es un lenguaje que complementa al tipado antes mencionado HTML y principalmente sirve como un idioma que nos permite darle una mejor interfaz a nuestra página web desde el uso de colores, tipo de fuente, tamaños, etc.

Una vez entendiendo el concepto del lenguaje CSS, el maestro nos pidió analizar la interfaz de Gmail para en parte comprender que o cuantas cajas conlleva la pagina, ademas de el cuidado uso de colores que creo fue visto desde un lado más teorico. El modelo de caja en CSS define figuras rectangulares, incluyendo su relleno y margenes.

Como usar CSS

Anteriormente habiamos visto como funcionaba CSS. En esta clase aprendimos y aplicamos algunas de sus funciones. Algunas de ellas fueron: cambiar el tamaño de la fuente, asociar enlaces de internet (o referencias), cambiar el color de los parrafos; lo que es un selector, propiedad y valor de propiedad. Cómo crear bordes, cómo seleccionar varios elementos para al final tener una idea de cómo es crear un diseño para una página web.

CSS se maneja de selectores, que escogen ciertas etiquetas que incluimos en el HTML, en incluso clases que pueden tener esas mismas etiquetas. Por ejemplo: el header, body, h1, p, etc. Los atributos de cada etiqueta se ponen dentro de llaves y a traves de ellas es cuando puedes diseñar el tamaño de letra, fuentes, agregar bordes y distintos tipos de herramientas que ayudaran a darle un enfoque más detallado a tu pagina web.

Conocimos una página web que se llama "Bootstrap" y sirve principalmente como apoyo para la creación de una pagina web.

Recomendaciones sobre Diseño Web

En clases, el profe paso a 5 personas a mostrar mediante conferencia el diseño de su página web y a base de ello darnos algunas sugerencias. Algunas que recuerdo son que: no debemos usar colores que sean muy agobiantes para la vista (colores muy fuertes), usar un tipo de imagenes que tengan fondo transparente, darle un toque más serio como si lo fueramos a mostrar a alguna empresa por ejemplo, no tener los parrafos centrados sino justificados, que no se salgan del margen, usar un tamaño de fuente más apto para el usuario; que no sea imposible de leer. Tener un enfoque más minimalista en cuanto a diseño se trata: no poner una gran variedad de colores o imagenes, ni cosas que llenen exageradamente de detalles, sino que se vea simple y agradable a la vista.

JavaScript cuenta con un tipado debil. Se refiere a que es muy permisivo en cuanto a las variables declaradas y sintaxis.

En clase vimos sobre cómo se elabora un sistema comenzando con ideas que se llevan a cabo antes de plantear un proyecto. Nos explico los pasos que siguen después de un login, que es cuando el usuario no solo se crea una cuenta, sino también un identificador. De manera tecnica se le llama "formulario" pues el proceso de un registro es a base de responder preguntas y desde html hemos comenzado a elaborar uno con los requisitos que usualmente suele pedir tales como usuario, contraseña, email... Lo más fundamental al momento de codificar un login es que tiene que ser lo más sencillo y entedible posible para la comprension del usuario siendo esta lo más explicita posible.

PHP

PHP es un lenguaje de programacion que funciona desde el lado del servidor y se adapta especialmente al desarrollo web. El puente PHP conecta al HTML y a su vez a la base de datos. Para que el cliente pueda hacer una consulta al base de datos, el cliente necesita conectarse a un servidor, el cual consulta al PHP y este ultimo consulta a la base de datos. Su modo de funcionamiento se basa en que, desde el propio codigo HTML se incrustan scripts de PHP. El tipo de pantallas donde se haria uso del lenguaje PHP serian por ejemplo: Recuperar Contraseña, Registro, Login y probablemente el Menu.
